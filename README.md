Trabalho Prático: Tecnologias Web 2025/26

Este repositório contém o desenvolvimento do trabalho prático realizado no âmbito da unidade curricular de Tecnologias Web do Instituto Superior de Engenharia de Coimbra (ISEC).

O objetivo principal deste projeto é a implementação prática de conceitos de HTML, CSS e JavaScript, replicando a interface de uma plataforma de Rent-a-car e desenvolvendo um motor de pesquisa de veículos dinâmico.
🚀 Funcionalidades
1. Réplica de Interface (HTML & CSS)

    Design Fiel: Réplica do website da Sixt efetuada exclusivamente com base em HTML/CSS.

    Layout Moderno: Utilização obrigatória de CSS Flexbox e CSS Grid.

    Responsividade: Implementação adaptável a diferentes resoluções através de um máximo de 3 media queries/breakpoints.

    Componentes Customizados: Inclui um sandwich menu para resoluções móveis e submenus com efeito hover na barra de navegação.

2. Motor de Pesquisa (JavaScript)

    Formulário Dinâmico: Permite selecionar a categoria (Carro/Carrinha), local de levantamento e datas.

    Lógica Condicional: O campo "Local de Devolução" é exibido apenas se a opção "Devolução em Local Diferente" for selecionada.

    Geração de Conteúdo: Os veículos são gerados via JavaScript.

    Cálculo de Preços:

        Preços diferenciados por categoria e dia da semana/fim de semana.

        Aplicação de tarifa empresarial com descontos progressivos (10% a 20%) conforme a duração.

        Taxa adicional de 30€ para devoluções em locais distintos.

        Visualização detalhada do cálculo do preço total.

🛠️ Tecnologias Utilizadas

    HTML5: Estruturação semântica de raiz (sem templates).

    CSS3 (SASS): Estilização e responsividade (sem frameworks como Bootstrap).

    JavaScript (ES6+): Manipulação do DOM e lógica de cálculo.

📁 Estrutura do Projeto

O projeto está organizado nos três ficheiros fundamentais exigidos:

    index.html: Estrutura da página principal.

    estilos.css: Folha de estilos (gerada a partir de SASS).

    logica.js: Implementação de todas as funcionalidades.
